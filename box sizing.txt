Що таке box-sizing?
box-sizing — це властивість CSS, яка контролює, як обчислюються ширина та висота елемента. Вона визначає, що включено у загальні розміри елемента.

Два основні значення:
content-box (за замовчуванням):

У цьому режимі ширина і висота визначаються тільки для вмісту (тексту, зображень тощо).
Padding (внутрішні відступи) і border (рамка) не враховуються в ширині та висоті. Це означає, що фактичний розмір елемента буде більшим, ніж задані width і height.
Приклад:

Якщо ви задаєте ширину 300px, а padding і border становлять 45px (15px border + 30px padding з кожної сторони), то фактична ширина буде 300px + 45px = 345px.
border-box:

У цьому режимі ширина і висота включають в себе вміст, padding і border.
Це означає, що якщо ви задаєте ширину 300px, то вся ширина (вміст + padding + border) залишиться 300px.
Приклад:

Якщо ви задаєте ширину 300px і padding 30px і border 15px, фактична ширина буде 300px, тому що padding і border враховуються в цю ширину.
Як це виглядає на практиці?
Розглянемо ваш приклад:

css
Copy code
.content {
    box-sizing: border-box; /* Змінюємо поведінку */
    width: 300px; /* Загальна ширина 300px */
    height: 150px; /* Загальна висота 150px */
    
    margin: 40px; /* Відступи навколо елемента */
    border: 15px solid blue; /* Рамка шириною 15px */
    padding: 30px; /* Внутрішні відступи 30px */
}
З border-box:
Загальна ширина буде 300px. Це означає, що вміст, padding і border сумарно не перевищать 300px.
Вміст всередині елемента буде меншим, адже padding та border враховані в загальну ширину.
Чому це важливо?
Зручність: border-box робить управління розмірами елементів більш передбачуваним, оскільки вам не потрібно додавати padding і border до ширини або висоти.
Легкість в дизайні: З border-box простіше розрахувати макет сторінки, особливо при роботі з флекс-контейнерами або сітками.
Висновок
box-sizing: content-box; — значення за замовчуванням, ширина і висота тільки для вмісту.
box-sizing: border-box; — включає padding і border в ширину і висоту, що робить розміри елемента більш зрозумілими.
